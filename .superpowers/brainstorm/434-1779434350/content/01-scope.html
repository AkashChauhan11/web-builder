Which Elementor-style capabilities matter most?

Pick all that you want in MiniPress. You already have basic block drag/drop, 4 custom blocks, asset uploads, multi-locale, and SEO. These are the gaps toward "Elementor-like." Click each one you want.

1

Section → Column → Widget layout

Elementor's core primitive: drag a Section, split into 1-6 columns, drop widgets into columns. Every page is built from sections. Currently you have flat blocks.

2

Expanded widget library (20-40 widgets)

Heading, Text, Image, Button, Icon, Icon Box, Divider, Spacer, Video, Accordion, Tabs, Toggle, Counter, Progress Bar, Testimonial, Pricing Table, Image Carousel, Image Gallery, Social Icons, Google Map, Form, Alert, Star Rating, HTML, Shortcode.

3

Rich style panel (Content / Style / Advanced tabs)

Per-element: typography (font family, size, weight, line-height, letter-spacing), color, background (color/gradient/image), border, border-radius, padding/margin (4-side), shadow, hover state, transition, custom CSS. Elementor's right sidebar is its killer feature.

4

Per-breakpoint responsive values

Switch to Tablet/Mobile in the toolbar and change padding/font-size/visibility just for that breakpoint. Values inherit from larger breakpoints. Currently you have responsive preview only, not per-breakpoint overrides.

5

Inline WYSIWYG text editing

Click any heading or text on the canvas and type directly. No popup form. Elementor's biggest UX win over GrapesJS's default.

6

Template library (page + block templates)

"Insert template" button shows a grid of pre-designed pages (About, Pricing, Contact) and individual blocks (Hero variants, Testimonial layouts). One click inserts. Save your own pages as templates too.

7

Global styles (typography + color palette)

Set 4-6 site colors (Primary / Secondary / Text / Accent) and 4 text styles (H1-H4, Body). Every widget pulls from these by default. Change once, updates everywhere.

8

Motion / entrance animations

Fade-in, slide-up, zoom, on-scroll reveal, hover effects, parallax. Mostly visual polish, not core layout.

Tip: items 1, 2, 3 are the foundation. Items 4, 5, 6, 7 are the productivity wins. Item 8 is polish. Pick what you actually want — it's fine to start small.
